# 一机一中走班排课工具
使用electron做了一个Desktop App。

之前想要搞成在线自助选课系统，但是想到到还需要花钱租服务器、还需要教小盆友们怎么操作，心里就呵呵了。于是下定决心把排课的算法彻底搞了出来，目前在保证课程设置合理的情况下能达到99.7%的成功排课率。
